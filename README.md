# Todo Flask App

This is a simple To-Do List web app built with Flask.

## Features

- User registration and login
- Add, update the status, and delete tasks
- Each user sees only their tasks
- Responsive design using HTML/CSS

## Technologies Used

- Python
- Flask
- Flask-Login
- SQLAlchemy
- HTML / CSS / JS
